# YouTube Scroll Saver

Tired of losing your spot on YouTube when using the mini-player or navigating back? **YouTube Scroll Saver** automatically remembers your scroll position on main pages, search results, and user profiles. When you minimize a video, your previous scroll position is restored smoothly, letting you continue browsing without starting from the top. It also remembers your last watched video for quick navigation.

---

## Features

- Saves scroll position on YouTube homepage, search results, and user profiles.
- Restores scroll smoothly when navigating back or minimizing videos.
- Keeps track of the last watched video.
- Easy toggle ON/OFF via the extension popup.

---

## Tech Stack

- TypeScript
- Chrome Extension APIs
- Tailwind CSS
